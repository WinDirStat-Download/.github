## Quick Start with WinDirStat Download

1. **Get your WinDirStat Download** — Grab the portable `.exe` (no installation) or the installer version.
2. **Launch** — Double‑click `WinDirStat.exe`. The main window opens in under one second.
3. **Select what to scan** — Choose a drive (C:\, D:\, or a specific folder). Your **WinDirStat Download** starts scanning immediately.
4. **Read the treemap** — Each colored rectangle is a file. Larger rectangles mean larger files. Colors represent file types (green = documents, blue = videos, red = executables).
5. **Clean up** — Right‑click any treemap block → “Delete” or “Open in Explorer”. Your **WinDirStat Download** deletes directly or moves to Recycle Bin.
6. **Adjust (optional)** — Change color schemes, ignore system folders, or set scan exclusions. Most people use **WinDirStat Download** defaults.
7. **Work normally** — Close the **WinDirStat Download** after cleanup. No background services remain.
